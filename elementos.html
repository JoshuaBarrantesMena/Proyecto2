El Correo Electrónico

Elementos

  • Remitente: Es la persona o sistema que envía el correo electrónico a un destinatario.
  • Destinatario: La persona o grupo que recibe el mensaje de un remitente.
  • Asunto: Un título o breve descripción de lo que trata el contenido del correo.
  • Cuerpo del mensaje: El contenido principal, que puede incluir texto, imágenes y enlaces.
  • Archivos adjuntos: Cualquier tipo de documento que acompaña al mensaje del correo.

Formato

El correo electrónico está estructurado de una forma estándar que permite al usuario y el dominio del correo, todas las direcciones de correo electrónico están estructurados como usuario@dominio.com.

El cómo se utiliza cada elemento de la estructura es se definen de la siguiente manera:

  1. Usuario: Identifica a la persona o cuenta que es propietaria del correo, no se limita a personas, también a empresas o departamentos.
  2. @: Separador que indica el uso del protocolo de correo, se utiliza de forma estándar para separar el usuario del dominio al que pertenece el servicio del correo.
  3. Dominio: Representa al proveedor del servicio que es propietario del dominio, como Google o Microsoft.
